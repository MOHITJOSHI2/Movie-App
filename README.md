## Movie App
### This is a simple movie app that shows a list of movie which you can add to favourite and remove from favourite and aslo there is functionality for movie search. I am using "omdbapi" api to get the movie data.

## Technologies used
* React + Vite

## Installation and Working
* First clone the repo into your directory
* After cloning the repo go into the directory by writing "cd Movie-App" in terminal
* Then in terminal write "npm i" to install necessary library
* then finally run "npm run dev" and a link will come go to the link and that's it
